Como criar aplicativos móveis com JavaScript?

Aplicativos móveis baseados em JavaScript são mais comuns do que você imagina. Apps famosos, como o do Facebook e do Instagram, foram criados pela linguagem e tem apresentado um ótimo desempenho.

Se você também deseja criar seus aplicativos com JavaScript, o caminho mais fácil e rápido é através de um framework.

Neste artigo vamos apresentar os dois frameworks mais utilizados para criar apps móveis, bem como suas principais vantagens e desvantagens.

No final, indicarei algumas boas fontes para estudar e dominar o desenvolvimento de Aplicativos Móveis com JavaScript.

Pré-Requisitos

A parte visual do do aplicativo será construido com HTML e CSS. A programação será feita com JavaScript. Logo, é importante ter conhecimento dessas três linguagens.

Recomendo que tenha um bom conhecimento em JavaScript antes de aprender qualquer framework.

Um livro que costumo indicar é o You Don’t Know Javascript (grátis). Revise e veja se já conhece todos os conceitos. Se não conhecer, aprenda e só então siga adiante.

Uma vez que tenha dominado essas linguagens, vamos para os nossos frameworks.

React Native

O primeiro framework que abordaremos (e mais famoso) é o React Native.

Desenvolvido pelo Facebook, este framework é utilizado por grandes empresas, como Instagram, Airbnb, Wallmart e UberEats.

Você cria seu aplicativo utilizando HTML, CSS e JS e depois compila para linguagem nativa, ou seja, Java (Android) ou Swift (iOS).

Antes de começar a usar o React Native é preciso aprender React, uma biblioteca também criada pelo facebook. A curva de aprendizagem é relativamente curta se você aprendeu bem o JavaScript.
